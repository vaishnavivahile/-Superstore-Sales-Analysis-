# Superstore Sales Analysis

## Overview
An end-to-end Data Analytics project analyzing US Superstore Sales data 
using Python and Power BI.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn) — EDA & Data Cleaning
- Power BI — Interactive 3-page Report

## Key Insights
- Sales grew 51% from 2014 to 2017
- Furniture has highest sales but lowest profit margin
- Discounts above 40% consistently lead to losses
- Central region underperforms due to high avg discount rate (24%)

## Project Structure
- `SuperstoreSales.ipynb` — Python EDA notebook
- `superstore_sales_analysis.pbix` — Power BI Report
- `superstore_clean.csv` — Cleaned dataset
- `Screenshots/` — Report screenshots
